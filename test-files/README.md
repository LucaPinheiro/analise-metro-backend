# Arquivos de Teste

Esta pasta contém arquivos de exemplo para testar a aplicação.

## Estrutura Recomendada

```
test-files/
├── modelo.ifc          # Arquivo BIM de exemplo
├── foto1.jpg           # Foto de teste 1
├── foto2.jpg           # Foto de teste 2
├── foto3.jpg           # Foto de teste 3
└── README.md           # Este arquivo
```

## Como Obter Arquivos de Teste

### Arquivos BIM

Você pode usar qualquer arquivo `.ifc`, `.dwg` ou `.obj` válido. Exemplos:
- Arquivos de exemplo do Revit
- Modelos de exemplo do Blender exportados como `.obj`
- Arquivos de teste do CloudCompare

### Fotos

Use fotos reais de canteiro de obras ou fotos de teste:
- Mínimo: 3 fotos
- Recomendado: 10-20 fotos
- Formato: JPG ou PNG
- Tamanho mínimo: 100KB por foto
- Tamanho máximo: 100MB por foto

## Nota

Estes arquivos são apenas para testes locais. Não commite arquivos grandes no repositório.
